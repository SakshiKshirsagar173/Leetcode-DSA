# LeetCode 2075 – Decode the Slanted Ciphertext

## 🧩 Problem Description
A string `originalText` is encoded using a **slanted transposition cipher** into `encodedText` with a fixed number of rows.

- Characters are first placed diagonally (top-left → bottom-right) into a matrix.
- Empty cells are filled with spaces `' '`.
- The encoded string is generated by reading the matrix **row-wise**.
- Given `encodedText` and `rows`, we must reconstruct `originalText`.

## 💻 Java Code Implementation

```java
class Solution {
    public String decodeCiphertext(String encodedText, int rows) {
        int cols = encodedText.length() / rows;

        char[][] mat = new char[rows][cols];
        int i = 0, j = 0;

        // Fill the matrix row-wise
        for (int ind = 0; ind < encodedText.length(); ind++) {
            mat[i][j] = encodedText.charAt(ind);
            if (j < cols - 1) {
                j++;
            } else {
                i++;
                j = 0;
            }
        }

        StringBuilder sb = new StringBuilder();

        // Read the matrix in slanted (diagonal) order
        for (j = 0; j < cols; j++) {
            int slant = 0;
            for (i = 0; i < rows; i++) {
                if (j + slant < cols) {
                    sb.append(mat[i][j + slant]);
                    slant++;
                }
            }
        }

        return sb.toString().stripTrailing();
    }
}
